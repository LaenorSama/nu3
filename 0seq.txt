по топ 5 методов:
1. списки (list):
* s1.append(s2) - добавляет элемент в конец списка, если мы подадим другой список то он станет одним элеменотом первого списка,
* s1.extend(s2) - добавляет элементы второго списка в конец первого списка,
* s1.copy() - копирует список, s1.sort() - сортирует список по алфавиту,
* s2 = s1.pop([i]) - удаляет элемент с индексом i и возвращает его, очень удобно. получаем 2 дейсвтия в одном

2. словари (dict):
* d1.copy() - копирует словарь,
* d1.items() - возвращает пары (ключ, значение),
* d1.update(d2) - обновляет словарь данными из d2 (d2 это список кортежей)
* d1.keys() - возвращает ключи в словаре
* d1.values() - возвращает значения в словаре

3. кортежи (tuple)
* t1.count(element) - возвращает количество вхождений такого элемента
* t1.index(element) - возвращает индекс первого встретившегося такого элемента
* в документации только 2 метода

4. множества (set)
* len(s) - количество элементов в множестве
* s.isdisjoint(s2) - проверяет есть ли общие элементы множества s и s2(тут может быть не множествао а список или кортеж или словаь(сверит по ключам), выдаст True если общих элементов нет
* s.copy() - копирует множество
* s.discard(element) - удаляет элемент из множества, если он там есть
* s.add(element) - добавляет элемент в множество

5. строки (str)
* len(s) - длина строки
* s.split(sep=' ') - разделяет на подстроки по разделителю
* s.isalpha() - проверка состоит ли строка их букв
* s.lower() - переводит в нижний регистр
* s.count(s2) - сколько раз s2 встречается в s
